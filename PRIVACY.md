# Privacy — plain summary

The rendered version of this notice lives at `/privacy` on the deployed site
(`src/pages/Privacy.tsx`). This file is the plain-text mirror.

- This site collects **no personal data**.
- It uses **no cookies**.
- It uses **no analytics, tracking, fingerprinting, or third-party scripts**.
- The only browser-storage value it writes is a `localStorage` entry named
  `koji-theme` that remembers the light / grey / dark theme preference.
- All requests are same-origin. No fonts, images, or scripts are loaded from
  third-party CDNs.
- Clicking the Discord invite navigates you to discord.com, which is a
  third-party service governed by its own privacy policy.
- The static host (GitHub Pages or Cloudflare Pages) may retain standard
  request logs under its own privacy policy; those logs are not accessible
  to this project.

## GDPR, CCPA, Texas DPSA

Because no personal data is collected, there is no lawful-basis analysis to
publish, no controller/processor relationship to disclose, no data-subject
request workflow to operate, and no consent banner to display. Any request
under GDPR, CCPA, or Texas DPSA can be answered truthfully: we hold no data
about you.

If you contact `hello@koji.coffee`, that email will be handled under the
email provider's own privacy terms.
